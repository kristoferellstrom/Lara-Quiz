# Frontend (React + Vite + TypeScript)

Kör lokalt:
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```
Standard-URL: http://localhost:5173

Miljövariabler (`.env`):
- `VITE_API_URL` – t.ex. `http://127.0.0.1:8000`

Lägg bilder i `public/media/` och referera dem som `/media/filnamn.jpg` i `questions.json`.
